# diagrams-svg-virtual-hom

Virtual-hom SVG rendering backend for `diagrams`. Mostly a clone of https://github.com/diagrams/diagrams-svg.

## How to build

```
stack setup
stack build
```

* To generate the haddocks, `stack --stack-yaml stack-ghc.yaml haddock`
